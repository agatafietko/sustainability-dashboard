# How to Format Card Visual (Title and Size)

## 🎯 **What "Format: Add title, adjust size" Means**

When you create a card visual showing "Total Publications", you want to:
1. **Add a title** - Give it a label like "Total Publications" so people know what it shows
2. **Adjust size** - Make it bigger/smaller to fit your dashboard layout

---

## 📋 **Step-by-Step: Format Your Card**

### Step 1: Select the Card Visual

1. **Click on the card** you created (the one showing "Total Publications" number)
2. You should see it highlighted/selected (with a border around it)

### Step 2: Open Format Pane

**Option A: Visual Format Pane**
1. Look at the **right side** of the screen
2. You should see a **"Format visual"** or **"Format"** pane
3. If not visible, click the **paint roller icon** or **"Format"** button (usually in Visualizations pane)

**Option B: Right-Click Method**
1. **Right-click** on the card visual
2. Select **"Format visual"** or **"Format"**

### Step 3: Add Title

1. In the **Format pane**, scroll down to find **"General"** or **"Card"** section
2. Look for **"Title"** option
3. **Turn it ON** (toggle switch) - it might say "Off", click to turn "On"
4. A **"Title text"** field should appear
5. **Type**: "Total Publications" (or whatever you want to call it)
6. The title appears above or below the number on your card

### Step 4: Adjust Size

**Method 1: Drag to Resize**
1. **Click and hold** on the edges or corners of the card
2. **Drag** to make it bigger or smaller
3. Release when it's the size you want

**Method 2: Using Format Pane**
1. In Format pane, look for **"General"** section
2. Find **"Width"** and **"Height"** fields
3. **Type numbers** (e.g., Width: 200, Height: 150)
4. Card resizes automatically

---

## 🎨 **Additional Formatting Options**

While you're formatting, you can also:

### Change Number Format
1. In Format pane, find **"Values"** or **"Data labels"** section
2. Look for **"Display units"** or **"Format"**
3. Choose format (Whole number, Decimal, etc.)

### Change Colors
1. In Format pane, find **"Card"** or **"Background"** section
2. Click **"Background color"** to change card color
3. Click **"Font color"** to change number color

### Change Font Size
1. In Format pane, find **"Values"** section
2. Look for **"Text size"** or **"Font size"**
3. Adjust slider or type number (e.g., 24, 36, 48)

## ✅ **Quick Checklist**

For "Total Publications" Card:
- [ ] Card is selected (has border)
- [ ] Format pane is open
- [ ] Title is turned ON
- [ ] Title text says "Total Publications"
- [ ] Card size is adjusted (big enough to read)
- [ ] Number is readable (font size is good)

---

## 💡 **Pro Tips**

1. **Consistent Sizing**: Make all 4 KPI cards the same size
2. **Same Title Style**: Use consistent formatting for all card titles
3. **Large Numbers**: Make the number font bigger (24-36pt) so it's prominent
4. **Clear Labels**: Use descriptive titles so viewers know what they're looking at

---

## 🔧 **Where to Find Format Options**

**Format Pane Sections:**
- **General**: Size, position, title
- **Card**: Background, border, padding
- **Values**: Number format, font size, color
- **Title**: Title text, font, position

**Look for these toggles/options:**
- Title: **On/Off** toggle
- Title text: Text box
- Width/Height: Number fields
- Font size: Slider or number field
