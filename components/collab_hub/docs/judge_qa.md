# Judge Q&A Sheet

## Is this AI? How accurate is it?
This is a transparent, rule‑based scoring system, not a predictive AI model. We prioritized explainability so decision‑makers can see why a match is recommended.

## Why is career fit only 15%?
Topic alignment and methodological complementarity drive research synergy most strongly. Career stage matters for mentorship, but should not dominate collaboration quality.

## Where did the data come from?
From the provided publications dataset. No external scraping was used.

## How can we see who is paired?
Each row is a specific researcher pair. Selecting a row displays the score breakdown and a natural‑language explanation.

## What is new compared to the status quo?
Existing dashboards show counts and trends. The Collaboration Hub recommends actionable collaborations and explicitly rewards complementary methods, not just similarity.

## Is the demo data real?
The demo dashboard uses `CCS_Demo_Data.csv`, which is generated from real profiles with small randomized variation to demonstrate the logic clearly.
