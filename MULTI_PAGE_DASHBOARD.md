# Multi-Page Dashboard - Organizing All Your Visuals

## 🎯 **Solution: Create Multiple Pages**

Instead of cramming everything on one page, create **multiple pages** for different sections:

- **Page 1: Overview** - Key metrics and trends
- **Page 2: SDG Explorer** - SDG coverage and analysis
- **Page 3: Department Details** - Department performance
- **Page 4: Impact Stories** - Dynamic impact narratives

---

## 📋 **Step 1: Add New Pages**

### How to Add a Page:

1. **Look at the bottom** of your report canvas
2. You'll see **"Page 1"** tab
3. **Click the "+" icon** next to "Page 1"
   - Or right-click "Page 1" → **"New page"**
   - Or: **Insert** → **"New page"**
4. A new page tab appears: **"Page 2"**
5. **Double-click the page tab** to rename it (e.g., "SDG Explorer")

---

## 🎨 **Step 2: Make Visuals Smaller to Fit**

### Resize Current Visuals:

1. **Click on a visual** (e.g., your line chart)
2. **Grab the edges or corners** (you'll see resize handles)
3. **Drag inward** to make it smaller
4. **Release** when it's the size you want

### Make Multiple Visuals Fit:

**Option 1: Side-by-Side**
- Line chart: Left half
- Bar chart: Right half
- Both same height

**Option 2: Stack Vertically**
- Line chart: Top half
- Bar chart: Bottom half
- Both full width

**Option 3: 2x2 Grid**
- Top row: 2 cards side-by-side
- Bottom row: 2 charts side-by-side

## 🔧 **Step-by-Step: Resize Your Current Visuals**

### Make Cards Smaller:

1. **Select each card**
2. **Drag edges** to make them narrower:
   - Width: ~200-250px (instead of 300-400px)
   - Height: ~150-180px
3. **Arrange in one row** - they should all fit now

### Make Line Chart Smaller:

1. **Select line chart**
2. **Drag bottom edge up** - reduce height to ~300px
3. **Keep full width** - or make it 50% width if you want side-by-side

### Arrange Bar Chart:

1. **Select bar chart**
2. **Place below line chart** (stack vertically)
3. **Or place to the right** (side-by-side, each 50% width)

## 🎯 **Quick Space-Saving Tips**

### 1. Reduce Padding
- **Format pane** → **"General"** → **"Padding"**
- Reduce to 5-10px (instead of 20px)

### 2. Smaller Fonts (If Needed)
- Card titles: 12-14pt (instead of 16-18pt)
- Chart titles: 16pt (instead of 18pt)

### 3. Remove Unnecessary Elements
- Hide axis titles if labels are clear
- Remove gridlines if they clutter
- Simplify legends

### 4. Use Space Efficiently
- **Side-by-side**: 2 visuals at 50% width each
- **Stack vertically**: Full width, less height
- **Compact cards**: Smaller cards in one row
