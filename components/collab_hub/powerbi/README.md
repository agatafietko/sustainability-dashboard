# Power BI – Collaboration Hub

## Demo dashboard
Load `CCS_Demo_Data.csv` to show the collaboration hub in action.  
This demo table already includes match scores, reasons, and researcher attributes.

## Full dashboard (optional)
Load:
- `Collab_Matches_For_PowerBI.csv`
- `Researcher_Profiles_For_PowerBI.csv`

Create relationships on researcher names or IDs to enable filtering by department, method, and SDG.

## Usage
Filter by SDG, method, and career stage.  
Select a row to display the “Why this match?” explanation.
