# Data Policy

We do not publish the raw publications dataset in this repository. The dataset was provided by the case competition and may be subject to confidentiality or usage restrictions.

What we include:
- Documentation of data fields and processing steps
- Methodology and scoring logic
- Scripts that generate outputs

What we exclude:
- Raw publication data (CSV/XLSX)
- Power BI `.pbix` files

If access to the original dataset is authorized, you can reproduce outputs by placing the input file in the expected location and running the scripts.
