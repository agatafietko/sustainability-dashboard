## 🎯 **PROFESSOR'S CRITICAL CRITERIA (Your Foundation)**

Based on the stakeholder interview, the professor identified **three critical factors** for choosing research collaborators:

1. **Research Topic** (Primary) - Shared research interests
2. **Method** (Critical) - **Complementary** skill sets (different methods)
3. **Career Stage** (Strategic) - Strategic fit (e.g., pre-tenure with post-tenure for mentorship)

**Key Insight**: The professor emphasized seeking **complementary** (different) methods, not identical methods!

## 📋 **STEP-BY-STEP IMPLEMENTATION GUIDE**

- [ ] Build user profile form capturing:

- [ ] Store profiles in database
- [ ] Link to existing publication data (FAISS, SDG scores)

**Key Focus**: Make "complementary methods" explicit in the form

---

#### **1.2 Implement Compatibility Score Algorithm**
- [ ] Implement Topic Match (50% weight)
  - [ ] FAISS similarity search
  - [ ] SDG alignment matching
  - [ ] Keyword overlap calculation
- [ ] Implement Method Match (35% weight)
  - [ ] Method classification system
  - [ ] Complementarity scoring (different = higher score)
  - [ ] **CRITICAL**: Verify that complementary methods score higher
- [ ] Implement Career Stage Match (15% weight)
  - [ ] Career stage extraction
  - [ ] Strategic fit calculation
  - [ ] Mentorship opportunity detection

**Key Focus**: Method Match must reward DIFFERENT methods, not same methods

- [ ] Search by SDG alignment
