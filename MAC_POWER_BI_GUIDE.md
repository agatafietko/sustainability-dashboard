# Power BI on Mac - Your Options

## ⚠️ Important: Power BI Desktop is Windows-Only

Power BI Desktop doesn't have a native Mac version. Here are your best options:

---

## 🎯 **OPTION 1: Power BI Service (Web) - Recommended for Mac**

Since you're on Mac, let's use **Power BI Service (web)** with a workaround for CSV imports.

### Step-by-Step for Mac:

#### **Step 1: Upload CSVs to OneDrive**

1. Go to https://onedrive.live.com (or use OneDrive app)
2. Sign in with Microsoft account (free)
3. Create a folder: "Power BI Dashboard"
4. Upload your 4 CSV files:
   - `sdg_lookup.csv`
   - `Publications_cleaned.csv`
   - `Keywords_cleaned.csv`
   - `SDG_Mappings_cleaned.csv`

#### **Step 2: Create Dataset in Power BI Service**

1. Go to https://app.powerbi.com
2. Sign in (same Microsoft account)
3. Click **My Workspace** (left sidebar)
4. Click **+ New** → **Dataset**
5. Choose **OneDrive for Business** or **Files**
6. Navigate to your folder
7. Select **Publications_cleaned.csv** first
8. Click **Create**

#### **Step 3: Create Report**

1. After dataset loads, click **Create Report**
2. You'll see the report canvas

#### **Step 4: Add More Data Sources**

⚠️ **Important**: Power BI Service has limitations with multiple CSV relationships.

**Workaround**: Combine data first, or use separate datasets

**Better approach**: Use Excel Online (free, works on Mac)

---

## 🎯 **OPTION 2: Excel Online (Free, Mac-Friendly) - BEST FOR MAC**

Excel Online can create dashboards and works perfectly on Mac!

### Step 1: Upload to OneDrive

1. Upload your CSV files to OneDrive (same as above)

### Step 2: Open in Excel Online

1. Right-click CSV in OneDrive → **Open in Excel Online**
2. Or go to https://www.office.com → Excel

### Step 3: Create Power Pivot / Pivot Tables

1. **Insert** → **PivotTable**
2. Create relationships between tables
3. Build charts and visuals
4. Can create interactive dashboards

### Step 4: Export / Share

- Export to PDF
- Share link
- Save as Excel file

---

## 🎯 **OPTION 3: Tableau Public (FREE, Native Mac App) - HIGHLY RECOMMENDED**

Tableau Public is **free**, **native Mac app**, and **perfect for case competitions**!

### Download Tableau Public for Mac:

1. Go to https://public.tableau.com/app/download
2. Download **Tableau Public** (Mac version)
3. Install (drag to Applications)
4. Create free account

### Import Your Data:

1. Open Tableau Public
2. **Connect** → **Text File**
3. Select your CSV files
4. Create relationships
5. Build dashboards

### Advantages:
- ✅ Native Mac app
- ✅ Free
- ✅ Professional dashboards
- ✅ Easy CSV import
- ✅ Great for competitions
- ✅ Can publish online

### Disadvantage:
- ⚠️ Different tool (but similar concepts to Power BI)

---

## 🎯 **OPTION 4: Google Data Studio (Free, Web-Based)**

Completely web-based, works on Mac, free!

### Steps:

1. Go to https://datastudio.google.com
2. Sign in with Google account
3. **Create** → **Data Source**
4. Upload CSV files (or connect from Google Drive)
5. Create dashboards
6. Share link

### Advantages:
- ✅ Free
- ✅ Web-based (no download)
- ✅ Easy to use
- ✅ Great sharing features

### Disadvantages:
- ⚠️ Less powerful than Power BI
- ⚠️ Different interface

---

## 🎯 **OPTION 5: Virtualization (Advanced)**

If you really need Power BI Desktop:

### Parallels Desktop (Paid ~$100/year)
- Run Windows on Mac
- Install Power BI Desktop
- Full functionality

### VirtualBox (Free)
- Free virtualization
- Install Windows (need Windows license)
- More complex setup

### Boot Camp (Intel Macs only)
- Dual boot Windows
- Free (but need Windows license)
- Not available on Apple Silicon Macs
