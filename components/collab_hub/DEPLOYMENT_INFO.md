# Streamlit Cloud Deployment - Quick Reference

## 🔗 GitHub Repository

**Repository URL**: 
```
https://github.com/meryemrafiq14-hue/sustainability_case_competition.git
```

## 📍 App Location

**Main file path for Streamlit Cloud**:
```
components/collab_hub/app.py
```

## 🚀 Quick Deploy Steps

1. Go to: https://share.streamlit.io/
2. Sign in with GitHub
3. Click "New app"
4. Fill in:
   - **Repository**: `meryemrafiq14-hue/sustainability_case_competition`
   - **Branch**: `main`
   - **Main file path**: `components/collab_hub/app.py`
   - **App URL**: (choose a name)
5. Click "Deploy"

## 📋 Requirements

- Original CSV file must be in repository root: `for distribution case competition filtered_publications.csv`
- All dependencies in `components/collab_hub/requirements.txt`

## 🌐 Your App URL

After deployment:
```
https://[your-app-name].streamlit.app
```
