# Case Competition Submission Overview

## 📋 **Submission Structure**

This case competition submission presents the **Illinois Sustainability Impact Engine**, a comprehensive platform with **5 integrated components**.

### **Component Focus**

**⭐ Collaboration Hub** - **This is the component we focused on and fully documented.**

- Complete Python scripts with full logic
- End-to-end data pipeline documentation
- Complete methodology explanation
- Power BI integration guide
- Judge Q&A documentation
- Business logic and decision rationale

### **Other Components** (Integrated in Power BI)

The following components are part of the platform and integrated in the Power BI dashboard, but are not as extensively documented in this repository:

1. **Sustainability Dashboard** - SDG wheel visualization, coverage overview
2. **Research Coverage Analysis** - Gap identification, department-level analysis
3. **Impact Engine** - Impact scores and outcomes
4. **AI Prototype** - Natural language query interface

---

## 🎯 **Why Collaboration Hub Was Chosen**

We chose to focus on the Collaboration Hub because:

1. **Technical Depth**: Requires complex algorithm design (multi-factor scoring)
2. **Innovation**: Rewards complementary methods, not just similarity
3. **Transparency**: Rule-based system that's fully explainable
4. **Business Value**: Directly addresses collaboration challenges
5. **Demonstrable**: Can show end-to-end pipeline from data to visualization
